# 🧱 Application LEGOTherapie

Bienvenue sur le dépôt du projet **LEGOTherapie**, une application web conçue pour accompagner les professionnels (orthopédagogues, éducateurs) et les apprenants dans le cadre d'ateliers basés sur la manipulation et le jeu éducatif.

## 🎯 Objectifs du projet
* **Développement cognitif :** Stimuler les fonctions exécutives et la métacognition par le biais d'activités ludiques et concrètes.
* **Compétences psychosociales :** Favoriser la collaboration, la communication et la résolution de problèmes en groupe.
* **Apprentissage par l'erreur :** Utiliser la manipulation physique comme levier d'autorégulation et de remédiation pédagogique.

## 🚀 Technologies utilisées
* **HTML5 / CSS3 / JavaScript :** Pour une interface utilisateur fluide, accessible et interactive.
* **Web3Forms :** Pour la gestion simplifiée et sécurisée des formulaires de contact ou de retour d'expérience.
* **GitHub Pages :** Pour un déploiement instantané et un accès direct pour les stagiaires et partenaires.
